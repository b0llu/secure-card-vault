Common topics

Security Concerns
Found a potential vulnerability or security issue? Email directly. Please don't open a public issue — we'll work with you privately to address it.
Bug Reports
App crashing or something not working as expected? Please include your Android version, device model, and steps to reproduce. GitHub issues are preferred for bugs.
Feedback & Feature Requests
Have an idea to improve Card Vault? We'd love to hear it. Feature requests are best submitted as GitHub issues so they can be tracked and discussed openly.
Privacy Questions
Questions about what data the app stores or how encryption works? Our Privacy Policy covers this in detail, but feel free to ask directly.
Backup & Restore Help
Having trouble with export or import? Note that forgotten backup passwords cannot be recovered — this is by design. For other backup issues, reach out via email.
